## まずはモードを理解しよう

Vimを触ったことがない人が最初に戸惑うのが「**モード**」という概念や。

普通のエディタはキーボードを押したらそのまま文字が入力される。でもVimは違う。キーを押したとき、今どのモードにいるかによって動作が変わる。

MiniMaxを起動して `i` を押してみてな。画面の左下に `-- INSERT --` と表示されたら、それが「インサートモード」や。文字が入力できる状態になっとる。

`Esc`を押すと `-- INSERT --` の表示が消える。「ノーマルモード」に戻ったということや。

### 4つの主要モード

| モード | 表示 | 役割 | 移行方法 |
|---|---|---|---|
| **ノーマルモード** | （何も表示されない） | コマンド実行・カーソル移動 | `Esc` / `Ctrl-[` |
| **インサートモード** | `-- INSERT --` | テキスト入力 | `i` `a` `o` など |
| **ビジュアルモード** | `-- VISUAL --` | テキスト選択 | `v` `V` `Ctrl-v` |
| **コマンドラインモード** | `:` が表示される | ファイル操作・コマンド実行 | `:` |

:::message
「モードが分かれているせいでミスしやすい」という声をよく聞くけど、慣れると逆に便利や。ノーマルモードにいる間は誤って文字が入力されることがないので、安心してカーソルを動かせる。
:::

### ノーマルモードが基本

Vimは**ノーマルモードをデフォルト状態**として設計されとる。テキストを入力したいときだけインサートモードに入って、入力が終わったら`Esc`でノーマルモードに戻る。

このリズムを覚えることが、Vimを使いこなす第一歩や。

## キーバインドのヒントを活用しよう

MiniMaxには `mini.clue` というモジュールが入っとる。これはキーバインドのヒントを表示してくれる機能や。

ノーマルモードで `<Space>`（スペースキー）を押してみてな。少し待つと画面にポップアップが表示される：

```
<Space>
  b  Buffers
  f  Find / Pick
  g  Git
  l  LSP
  ...
```

これがLeaderキー（`<Space>`）から始まるコマンドの一覧や。`<Space>f`と続けて押すと、さらに詳しいヒントが出る。このヒントを見ながら操作すれば、コマンドを全部暗記しなくても使っていける。

## カーソルの基本移動

### hjklによる移動

Vimでは矢印キーでもカーソルを動かせるけど、`h j k l`の4キーを使うのが基本や。ホームポジションから手を動かさずに移動できるからや。

```
     k
     ↑
 h ← ● → l
     ↓
     j
```

```
h: 左に移動
j: 下に移動（Jを下向きの「J」に見立てて覚えよう）
k: 上に移動
l: 右に移動
```

最初はぎこちなく感じるかもしれんけど、1〜2週間使えば自然に指が動くようになる。

### 単語単位の移動

```
w: 次の単語の先頭へ（Word）
b: 前の単語の先頭へ（Backward）
e: 次の単語の末尾へ（End）
W: 次の単語（空白区切り）の先頭へ
B: 前の単語（空白区切り）の先頭へ
```

例：`hello world foo` という行で `w` を3回押すと `hello`→`world`→`foo`と移動する。

### 行内の移動

```
0: 行の先頭へ（数字のゼロ）
^: 行の最初の非空白文字へ
$: 行の末尾へ
```

### ファイル全体の移動

```
gg: ファイルの先頭へ
G:  ファイルの末尾へ
5G: 5行目へジャンプ
Ctrl-d: 画面の半分下にスクロール
Ctrl-u: 画面の半分上にスクロール
Ctrl-f: 1画面分下にスクロール
Ctrl-b: 1画面分上にスクロール
{: 前の空行（段落の先頭）へ
}: 次の空行（段落の末尾）へ
```

### 数字プレフィックス

移動コマンドの前に数字を付けると、その回数分実行される：

```
5j: 5行下に移動
10w: 10単語先へ移動
3}: 3段落先へ移動
```

これが使えるようになると移動が一気に速くなる。

## インサートモードへの入り方

テキストを入力するには、ノーマルモードから以下のキーでインサートモードに入る：

| キー | 動作 |
|---|---|
| `i` | カーソルの前に挿入（Insert） |
| `a` | カーソルの後に追加（Append） |
| `I` | 行の先頭に挿入 |
| `A` | 行の末尾に追加 |
| `o` | 現在行の下に新しい行を追加して挿入 |
| `O` | 現在行の上に新しい行を追加して挿入 |

この中で`i`、`a`、`o`の3つを覚えれば、最初はなんとかなる。

## 検索

### ファイル内検索

```
/pattern  : 前方検索（下方向）
?pattern  : 後方検索（上方向）
n         : 次の検索結果に移動
N         : 前の検索結果に移動
*         : カーソル下の単語を前方検索
#         : カーソル下の単語を後方検索
```

例：`/function` と入力してEnterを押すと、ファイル内の「function」を全て検索して最初のマッチに移動する。`n`で次の、`N`で前のマッチに移動できる。

### 行内検索

```
f{char}: 現在行でカーソルより右の{char}に移動
F{char}: 現在行でカーソルより左の{char}に移動
t{char}: fと同じだが、{char}の1文字前に移動
T{char}: Fと同じだが、{char}の1文字後に移動
;      : 最後のf/t/F/T検索を繰り返す
,      : 最後のf/t/F/T検索を逆方向に繰り返す
```

例：`console.log("hello")` という行で `f(` を押すと `(` の位置にカーソルが移動する。

:::message
mini.jumpモジュールにより、`f`/`t`の動作が強化されとる。複数行にまたがって検索できるようになっとるで。
:::

## ファイルの保存と終了

Vimあるある「終了できなかった」問題を解決しよう。

コマンドラインモード（`:`を押してから入力）での操作：

```
:w      保存（Write）
:q      終了（Quit）
:wq     保存して終了
:q!     保存せずに強制終了
:wqa    全バッファを保存して終了
```

ノーマルモードからのショートカット：

```
ZZ   :wqと同じ（保存して終了）
ZQ   :q!と同じ（保存せずに終了）
```

:::message
「`:q`を押したのに終了できない」という時は、未保存の変更があるから。`:q!`を使うか、`:w`で保存してから`:q`してな。
:::

## Undo / Redo

```
u        : Undo（直前の変更を取り消す）
Ctrl-r   : Redo（取り消した変更をやり直す）
U        : 現在行の全変更を取り消す
```

Vimのundoは階層構造になっとって、かなり細かく取り消せる。

## ビジュアルモードでテキスト選択

ビジュアルモードはテキストを選択するためのモードや：

```
v        : 文字単位の選択開始
V        : 行単位の選択開始
Ctrl-v   : 矩形（ブロック）選択開始
```

選択した後は`d`で削除、`y`でコピー、`c`で変更といった操作ができる。

**矩形選択の実用例：**

```
# 複数行のコメント先頭に「# 」を追加したい場合
Ctrl-v で行頭を複数行選択
I で行頭に挿入
"# " と入力
Esc で全行に適用
```

まず以下のキーを体に覚えさせてほしい：

**絶対必要な10キー：**
```

:w :q      保存・終了
/          検索
u          Undo
```

**次に覚えたい10キー：**
```

Ctrl-d u   半画面スクロール
* #        カーソル下の単語を検索
n N        検索結果を移動
```
